Regression test for 6735312

AppletContext.showDocument fails silently in IE7 if target contains hyphens

This test is intended to be run in IE on Windows, but should execute correctly in Firefox 3 as well. It should only be run with the new Java Plug-In.

In order for this test to run correctly, the pop-up it opens must be allowed. On IE you can allow it manually within the first ten seconds of running the test. On Firefox it appears the only way to allow the popup is to turn off the pop-up blocker while running this test.
