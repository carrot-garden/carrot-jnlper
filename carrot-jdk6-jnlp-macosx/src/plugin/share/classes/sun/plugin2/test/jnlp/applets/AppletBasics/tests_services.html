  1. SingleInstance Service 1 
    How to run SingleInstance service test ?

- Execute it in the same browser.

- Start the 1st instance in one tab.

        The console window shall make it visible,
        that this applet is waiting for another another trial of instantiation.
        You shall see the countdown!

- Start the 2nd instance (same URL) in another browser tab
        This applet shall never ever really start.
        Instead, trying to start it shall notify the first applet/instance
        that such an attempt has made, which shall lead to a successful test result.

You cannot start both instances at the very same time.
The first instance must have started completely,
before you can attampt to start the 2nd one !
